# Aumigo
Desenvolvimento de Projeto de um web site chamado "Aumigo". Esse site consiste em facilitar a adoção de Pets.

Esse Projeto utiliza Python-Flask como linguagem backend. HTML, CSS e JS para frontend. MongoDB para armazenamento de dados.
